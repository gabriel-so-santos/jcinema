# JCinema 🎬

A backend CRUD application built with Java and Spring Boot to manage a movie registry.
This project serves as a comprehensive starting point for learning the Spring ecosystem.

---

## Tech Stack
*   **Language:** Java 17+
*   **Framework:** Spring Boot (Spring Web, Spring Data JPA, Spring Security, Validation)
*   **Database:** PostgreSQL
*   **Migrations:** Flyway
*   **Build Tool:** Maven
*   **Utilities:** Lombok, Auth0 JWT
